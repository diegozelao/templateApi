## pt-br
## variaveis ocultas no projeto
Pegue os arquivos, na pasta "/envs" e coloque os arquivos na pasta raiz do projeto, renomeie os arquivos como '.env' por exemplo.

## iniciar
- npm install
- npm start

## no browser ou postman
- localhost:3000/
- localhost:3000/app
